# LocalGram 0.1.0

Initial release of LocalGram.

## Highlights

- in-app CameraX photo capture
- persistent hashtree block storage
- feed stored as a hashtree root
- Wi-Fi Aware nearby sync by root and missing block hashes
- automatic deduplication on receive
- feed-first Android UI with nearby connect/disconnect

## Notes

- Android-first release
- package name: `com.lauri000.nostrwifiaware`
- min SDK: 26
